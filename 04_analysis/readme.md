# 사용 가능 데이터
1. 각 논문 abstract

2. 각 논문 인용 수

3. 논문 검색시 사용한 keyword

4. 각 논문 내 효율화 라벨
| 라벨(bool) | 세부 내용 |
|---------------|----------------------------------------------------------------|
| 모델 사이즈 효율성 | 모델 효율성, 경량화, 스케일링 전략 등의 언급 여부                         |
| 모델 환경 영향    | 에너지 소비, 탄소 배출 등 환경 영향의 언급 여부                          |
| 제번스의 역설 언급 | 제번스의 역설에 대한 암묵적 언급 (효율화가 스케일 확장을 가능하게 함을 언급) 여부 |

5. 각 논문에서 제안하는 모델 또는 방법론
    - 파라미터 수
    - 수행 task
    - 성능
    - 사용 데이터 셋


# 분석 가능한 내용들 (CVPR)

## 논문 군집
- 각 논문 abstract를 이용한 topic modeling
- 논문 abstract 임베딩 벡터 분포 및 라벨 별 군집 위치 확인
    -> 각 논문은 검색 키워드가 중복될 수 있기 떄문에 keyword 별로 KDE 분포 곡선을 확인하면 됨
    -> 시간대 별로 비교하면 키워드 분포의 변화 양상을 확인할 수 있음

## 연구 트렌드
- 논문 인용수에 따른 3번 라벨 분포 확인
    -> 각 년도별로 인용수가 높은 논문은 어떤 라벨이 있을까?
    -> 각 년도별로 인용수가 높은 논문은 어떤 keyword를 가질까?

## 파라미터 증가 추이
- 시간대 별 파라미터 증가 추이 확인 (수행 task에 따라)
- 시간대에 따른 사용 데이터 셋 분포
- 시간대에 따른 성능 증가 추이
    -> 몇 % 성능 증가에 따라 얼마 만큼 파라미터 수가 증가 했는가?
- 파라미터 증가 수 추이와 효율성 언급 증가 추이 비교
    -> 효율성 기술이 변화에 비해 파라미터 수 증가는 얼마나 변화했는가?
